# Ronan's scrapbook for Megan

A responsive retro scrapbook built with plain HTML, CSS, and JavaScript.

## Preview locally

Open `index.html` directly or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

Pushing `main` triggers `.github/workflows/pages.yml`. The workflow builds a minimal public artifact containing only the webpage and the images it references; questionnaire answers, source notes, and duplicate photos are not published.
